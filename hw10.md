* Bubble Sort the list: 33, 56, 17, 8, 95, 22. Make sure the final result is from small to large.<br>
Write out the list after the 2nd pass.

>**33 56 17 8 95 22**<br>
>33 17 56 8 95 22 <br>
33 17 8 56 95 22 <br>
33 17 8 56 95 22 <br>
33 17 8 56 22 95 <br>
<br>
17 33 8 56 22 95 <br>
17 8 33 56 22 95 <br>
17 8 33 56 22 95 <br>
17 8 33 22 56 95 <br>

<br>

```c
#include<stdio.h>

int main(){
    int a[6] = { 33, 56, 17, 8, 95, 22};
    int n = 6, i = 0, j = 0, k = 0, temp = 0;
    
    for(i = 0; i < n; i++){
        for(j = 0; j < n-1; j++){
            if(a[j]>a[j+1]){
                temp = a[j];
                a[j] = a[j+1];
                a[j+1] = temp;
            }
            for(k = 0; k < n; k++){
                printf("%d ", a[k]);
            }
            printf("\n");
        }
        printf("\n");
    }
    
    return 0;
}

```
